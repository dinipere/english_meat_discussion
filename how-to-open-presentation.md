# How to open the presentation

## Windows

1. Open **PowerShell** or **Command Prompt**.
2. Go to the project folder:

```powershell
cd path\to\English_meat_discussion
```

3. Install everything once:

```powershell
npm install
```

4. Start the presentation:

```powershell
npm run dev
```

5. Open the link shown in the terminal, usually:

```text
http://localhost:3040/
```

## MacBook

1. Open **Terminal**.
2. Go to the project folder:

```bash
cd /Users/felix/Documents/GIT/English_meat_discussion
```

3. Install everything once:

```bash
npm install
```

4. Start the presentation:

```bash
npm run dev
```

5. Open the link shown in the terminal, usually:

```text
http://localhost:3040/
```

If `npm install` was already done before, only run `npm run dev`.
